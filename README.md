# bmw-scriptable-widgets

## 介绍
本程序是基于Scriptable开发的一个小组件。使用「小件件」作为开发框架而搭建。

主要用于从My BMW API中获取车辆相关信息并以小组件的形式展示到iOS设备上。

小组件本身不需要服务器支持。
